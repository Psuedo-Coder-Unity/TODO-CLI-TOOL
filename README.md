# TODO CLI Tool (C++)

A simple command-line todo list manager built in C++ using binary file storage.  
This project demonstrates file handling, structs, CLI argument parsing, and persistent storage.

---

## Features

- Add tasks from command line
- Display all tasks
- Mark tasks as completed
- Remove tasks by ID
- Clear all tasks
- Persistent storage using binary file (`List.dat`)

---

## Built With

- C++
- STL (`vector`, `fstream`, `filesystem`)
- Windows API (for executable directory handling)
- Binary file I/O
